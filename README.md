# Order Book
Order Book is a subsystem of an exchange. It is at its initial stage of development at the moment.

## Building It

This project uses Maven to build the project.

```
mvn package
```

To run the package:

```
java -jar target/orderbook-0.0.1-SNAPSHOT.jar
```
